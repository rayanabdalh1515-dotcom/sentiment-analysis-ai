# Sentiment Analysis AI Project

A simple AI project that predicts whether a sentence is positive or negative.

## How it works
- Uses a small dataset of sentences labeled as positive or negative
- Converts text into numbers
- Uses a Naive Bayes classifier for prediction

## How to run
1. Install libraries:
1. Install libraries:
   pip install -r requirements.txt

2. Run:
   python sentiment_analysis.py

3. Enter a sentence and the model will tell you if it's positive or negative.
## Summary

This is a simple AI project for text sentiment analysis. It predicts whether a sentence is positive or negative using a Naive Bayes classifier.
